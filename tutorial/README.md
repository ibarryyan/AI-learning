### 零基础入门 (8周)

| 周次 | 内容 | 资源 |
|:-----|:-----|:-----|
| 1-2 | Python 基础 | 廖雪峰 Python |
| 3-4 | 机器学习入门 | 吴恩达 ML 课程 |
| 5-6 | 深度学习基础 | PyTorch 官方教程 |
| 7-8 | 大模型入门 | Hugging Face 教程 |

### 进阶提升 (16周)

| 周次 | 内容 | 资源 |
|:-----|:-----|:-----|
| 1-4 | Transformer 原理 | CS224n, 论文精读 |

| 9-12 | RAG 系统 | LangChain |
| 13-16 | 项目实战 | Kaggle, 开源项目 |
